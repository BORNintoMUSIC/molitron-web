# SEO

## Search positioning

Molitron should rank through accurate, useful HTML content for commercial kitchen pollution control, exhaust filtration, and odor abatement. Nationwide capability leads; California and Denver/Colorado are focus markets rather than the primary headline.

## Page rules

- One descriptive H1 per indexable route.
- Unique title and meta description grounded in visible content.
- Canonical URL for every indexable page.
- Useful internal links from problems to products, applications, documentation, service, and quotes.
- Product and documentation pages must expose meaningful HTML rather than relying only on PDFs.
- Avoid keyword stuffing and unsupported geographic landing pages.

## Structured data

- Use Organization/manufacturer data that matches visible identity.
- Use Product data only for active MOAS and EPFA pages.
- Keep FAQ schema synchronized with visible FAQ content.
- Add BreadcrumbList only when visible breadcrumbs exist.
- Do not publish unsupported prices, ratings, reviews, or availability.

## Documents

- MOAS and EPFA documents are current public drafts pending separate revision projects.
- Enviro-Clean is discontinued and its documents are legacy/service material only.
- Keep legacy PDFs accessible for existing owners, but use `X-Robots-Tag: noindex` where practical so they do not compete with active products.
